Hướng Dẫn Vận Hành & Sử Dụng TeleDrive

Sau khi đã triển khai TeleDrive thành công, phần này sẽ hướng dẫn bạn cách sử dụng các tính năng cơ bản của nó.

Hoặc bạn có thể sử dụng web teledrive.top để sử dụng TeleDrive mà không cần tự triển khai.

1. Đăng Nhập Lần Đầu

Khi truy cập TeleDrive lần đầu, bạn sẽ cần đăng nhập bằng tài khoản Telegram của mình. Bạn chỉ cần nhập số điện thoại và nhận code qua ứng dụng Telegram trên điện thoại.

2. Giao Diện Chính

Sau khi đăng nhập, bạn sẽ thấy giao diện quản lý file cực kì đơn giản và dễ hiểu. Các danh mục chức năng được liệt kê bên phải và tổng số file được liệt kê ở cột bên trái

  • Cây thư mục: Hiển thị cấu trúc thư mục của bạn.
  • Danh sách file: Các file trong thư mục hiện tại.
  • Thanh công cụ: Các nút chức năng như Upload, Tạo thư mục mới, Xóa, Đổi tên...

3. Các Thao Tác Cơ Bản

3.1. Upload File

Để tải lên file:

  1. Click vào nút "Upload" (thường có biểu tượng mũi tên lên).
  2. Chọn file từ máy tính của bạn.
  3. Đợi quá trình tải lên hoàn tất.

3.2. Quản lý file

Các file được upload sẽ hiển thị thông tin chi tiết ở đây. Các chức năng như sửa tên file, xóa và chia sẻ nằm ở cột cuối bên trái

3.3. Xóa, Đổi Tên, Chia Sẻ File

Chọn nút chức năng ở cuối file để thao tác với file

Chúc bạn có trải nghiệm tốt với TeleDrive!
